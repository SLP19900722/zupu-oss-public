# Recording Script

这是一份更短的录屏操作稿，适合你开着微信开发者工具直接照着点。

## Before You Start

- 窗口切到手机模拟器视图
- 隐藏调试信息和多余面板
- 页面里先准备好演示数据
- 全程慢一点，每一步停 1 秒左右

## Click Path

1. 打开登录页，停 2 秒
2. 轻滑迁徙时间线，停 2 秒
3. 进入首页，停在影像轮播区
4. 左右切一张首页影像
5. 进入家谱树，缓慢拖动一次
6. 缩放一次家谱树，停 2 秒
7. 点开一个资料丰富的成员
8. 在成员详情页停留，露出职业、住址、简介
9. 返回，进入通知历史
10. 停在通知历史列表 2 秒
11. 回到首页或家谱树，作为结尾画面

## Voiceover Notes

- 开头：`这是一个面向单家族场景的数字族谱小程序社区版。`
- 首页：`除了树谱，它还把家族影像和故事入口放到了首页。`
- 家谱树：`树谱主视图可以直接浏览代际结构和配偶关系。`
- 成员详情：`成员详情页支持沉淀更完整的长期资料。`
- 通知历史：`如果家族活动需要提醒和留痕，也可以在这里查看通知记录。`
- 结尾：`仓库自带虚构演示数据，导入 SQL 后本地就能直接体验。`
