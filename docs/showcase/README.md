- `demo-cover.png`
- `01-login-timeline.png`
- `02-home-gallery.png`
- `03-family-tree.png`
- `04-member-detail.png`
- `05-notification-history.png`
- `06-select-self.png`
- `07-member-directory-overview.png`
- `08-member-directory-list.png`

如果你要把录屏成品一起留档，也可以临时放在这里，但不建议把体积很大的原始视频直接提交进 Git 仓库。
